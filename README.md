# Sidepieces

A web platform for developers, designers, and creators to showcase their side projects.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

This starts a local development server and opens the site in your browser.

## Project Structure

```
sidepieces/

├── src/
```
